# Maximum-Triplet-Sum

Given an array of n integers, find maximum triplet sum in the array Without duplications.

```
maxTriSum ([3,2,6,8,2,3]) ==> return 17 (6+8+3)
maxTriSum ([2,1,8,0,6,4,8,6,2,4]) ==> return 18 (8+6+4)
```
